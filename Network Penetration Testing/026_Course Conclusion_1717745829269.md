## Spiegazione approfondita

### 1. Host Discovery & Port Scanning
Coperto in 10+ video. Tool centrale: **Nmap**. Skills attesi:
- Identificare host attivi (`-sn`, `-PS`, `-PA`, `-PE`, `-PR` su LAN)
- Scelta del tipo di scan (`-sS`, `-sT`, `-sU`, `-sA`)
- Service version detection (`-sV`), OS detection (`-O`)
- Output saving (`-oA`) e import in Metasploit
- Stealth (timing T0-T1, fragmentation, decoy)

### 2. Service Enumeration
Skills attesi:
- Enumerare informazioni utili da servizi Windows: **SMB, NetBIOS, SNMP, RDP, MSSQL**
- Enumerare informazioni utili da servizi Linux: **Samba, SMTP, FTP, Finger, SSH**
- Sfruttare NSE protocol-specific (`smb-*`, `snmp-*`, `smtp-*`, `ftp-*`)
- Tool nativi: **enum4linux, smbclient, smbmap, rpcclient, snmpwalk, smtp-user-enum**
- Sapere **quando** usare un tool vs un altro (NSE rumoroso, tool dedicato più mirato)

### 3. Man-in-the-Middle / Network-based attacks
Skills attesi:
- **ARP spoofing** con `arpspoof` (bidirezionale)
- **DNS spoofing** con `dnsspoof`
- **NetBIOS NS / LLMNR poisoning** con **Responder**
- **SMB Relay attack** (Metasploit + ntlmrelayx)
- IP forwarding e setup MITM

### 4. Exploitation & Post-exploitation
Skills attesi:
- Exploit Windows: **SMB relay, MSSQL impersonation/xp_cmdshell, Rejetto HFS, BadBlue**
- Exploit Linux: **eGallery upload, Exim privesc, Shellshock, ProFTPD backdoor**
- Post-ex: **hashdump, Mimikatz/Kiwi, pivoting (autoroute, socks_proxy, portfwd, proxychains), RDP/SSH lateral**
- Credential dumping (NTLM, LSA, app installate via post modules)

### Cosa NON è coperto (rimando ad altri corsi)
- **Privilege Escalation** (Windows + Linux) → corso dedicato
- **Lateral Movement & Pivoting avanzato** (CrackMapExec, PsExec, WinRM, SOCKS over SSH, reGeorg) → corso dedicato
- **Active Directory attacks** → corso dedicato
- **Client-side / phishing** → corso dedicato
- **C2 frameworks** (Empire, Sliver, Cobalt Strike-like) → corso dedicato

### Filosofia di studio promossa
Setting di obiettivi → studio → revisione obiettivi raggiunti → tracciamento progresso. Modello applicabile non solo a questo corso ma a qualsiasi domain di pen testing.

## Punti d'attenzione per l'esame eCPPT

- **Formato 2024**: **45 domande multiple choice** su ambiente lab pratico (no report).
- **Aree con alta probabilità di apparire**:
  1. **Nmap completo**: flag scan, NSE, output, timing, evasion.
  2. **Enumeration SMB/SNMP** (null session, community string, NSE).
  3. **SMB Relay** (pre-req: signing disabled; tool Metasploit + ntlmrelayx).
  4. **MSSQL** (xp_cmdshell, impersonation, sysadmin role).
  5. **Hash dump + crack** (NTLM, hashcat modes, John format).
  6. **Pivoting Metasploit** (autoroute, socks_proxy, proxychains).
- **Concetti da ricordare**:
  - Porte: SMB 139/445, NetBIOS 137-139, SNMP 161 UDP, MSSQL 1433, FTP 21, Finger 79, SMTP 25.
  - Hashcat modes: 1000 NTLM, 3000 LM, 5500 v1, 5600 v2, 13100 TGS, 18200 AS-REP.
  - Comandi meterpreter chiave: `getsystem`, `migrate`, `hashdump`, `load kiwi`, `run autoroute`, `portfwd`.
- **Mind-set lab**: enum esaustivo PRIMA di exploitare, salva sempre risultati, lavora per workspace Metasploit.
- **Privesc e Lateral Movement avanzato** sono in altri corsi → ripassali separatamente per l'esame.
